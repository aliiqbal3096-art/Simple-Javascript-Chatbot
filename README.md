# Simple-Javascript-Chatbot
A beginner-friendly chatbot built using HTML, CSS, and JavaScript.   This project demonstrates DOM manipulation, event handling, and basic logic-based chatbot responses using JavaScript.
# Simple JavaScript Chatbot 🤖

This is a beginner-level chatbot project built using HTML, CSS, and JavaScript.

## 🚀 Features
- User input handling
- Bot responses using JavaScript conditions
- Dynamic DOM updates
- Chat history display

## 🧠 Concepts Used
- DOM Manipulation
- Event Handling (button click)
- JavaScript functions
- Conditional statements (if/else)
- createElement & appendChild

## 📌 Purpose
This project was built to practice DOM and understand how real chat applications work at a basic level.
